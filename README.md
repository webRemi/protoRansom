# protoRansom
Prototype for GhostHeist/GhostUtility

## Encrypt
Usage
```powershell
python ghostRansom.py <folder>
```
Example
```powershell
python ghostRansom.py 'H:\Financial datas\documents'
```

## Decrypt
Usage
```powershell
python ghostUtility.py <folder>
```
Example
```powershell
python ghostUtility.py 'H:\Financial datas\documents'
```
